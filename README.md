# learning-in-public

> "The story of working with something you can't fully trust — we trusted the calculator because it calculates. AI decides. This is learning to build with something that might go rogue."

---
 
Once, accountants did math by hand. The calculator didn't replace them — it made them unstoppable. AI is that moment again. Except this time, it decides. And that's what makes it harder to trust.

I tried to let AI build everything. Pure vibe coding.

It was exhausting.

Not because it was hard. Because I felt like a scout on someone else's project. Reading code I didn't write. Fixing bugs in logic I never understood. Nothing felt mine. Approving things I didn't understand. The AI was building. I was just watching.

So I stopped. And I asked a different question.

Not "can AI build this for me" — but "how do I work with AI without losing myself in it?"
We trust calculators because we understand what we're asking them to do. We know the math. We just don't want to do the arithmetic.

AI is different. It doesn't just calculate — it decides. And if you don't know what you're deciding, you're not the one building anymore.

This repo is me figuring that out. In public. Through a real product. With real mistakes.
 
---
 
## What's being built
A platform for architects, contractors, homeowners, and suppliers to get on the same page about building materials — especially products not available locally.
 
## What's being learned
How to orchestrate multi-agent systems and actually own the code they produce.
