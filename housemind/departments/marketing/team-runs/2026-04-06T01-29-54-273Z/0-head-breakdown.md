### 3. **Art & UI Designer**
**Objective:** Develop a high-fidelity UI design for the new annotation workspace that communicates the core features intuitively and supports user interaction in a mobile-first setup.

   - Create a layout for the product panel that dynamically filters based on active pins.

### 4. **User Experience (UX) Researcher**
**Objective:** Validate the requirements against user needs and provide recommendations for making the annotation workspace intuitive and delightful for all user personas.

3. Provide feedback on early mockups from **Art & UI Designer** for usability.
