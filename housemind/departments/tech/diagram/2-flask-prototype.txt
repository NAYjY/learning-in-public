┌─────────────────────────────────────────────────────────────────┐
│                    THE CORE INTERACTION LOOP                    │
│                                                                 │
│  1. Add image to project carousel (URL)                         │
│  2. Long-press on image → emoji menu                            │
│  3. Pick emoji → annotation pinned at that spot                 │
│  4. Double-click annotation → add product form                  │
│  5. Paste product URL → scrape images → pick one                │
│  6. Product linked to annotation (emoji type)                   │
│  7. Product grid shows products for current image's markers     │
│  8. Click annotation → its products sort to top                 │
│                                                                 │
│  image → pin → product → browse by pin                          │
│                                                                 │
│  THIS is what makes the app special.                            │
│  The Next.js app doesn't have any of this yet.                  │
